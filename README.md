## App for finding nearby lost dogs

### This mobile application was built with Ionic 4 and Firebase

## Purpose
### findme seeks to eliminate the lost dogs posts in the streets, implementing a friendly platform for dog enthusiasts and dog owners that can help them return the pets home.
